Welcome to the LeagueToolkit Wiki — your guide to the League of Legends modding ecosystem.

## What is LeagueToolkit?

LeagueToolkit is a collection of open-source tools for creating, managing, and installing mods for League of Legends. The ecosystem includes:

- **ltk-manager** — A desktop application for installing and managing mods
- **league-toolkit** — A Rust library for reading and writing League of Legends file formats
- **lol-meta-classes** — A toolchain for extracting and working with League's metaclass system

## Who is this wiki for?

- **Mod Users** — Learn how to install and manage mods using ltk-manager
- **Mod Creators** — Discover how to create custom skins, maps, and gameplay mods
- **Tool Developers** — Find technical references for file formats and APIs
- **Contributors** — Get involved with LeagueToolkit development
