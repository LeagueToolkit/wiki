Welcome to the LeagueToolkit Wiki - your guide to the League of Legends modding ecosystem.

## What is LeagueToolkit?

LeagueToolkit is a collection of open-source tools for creating, managing, and installing mods for League of Legends. The ecosystem includes:

- **[LTK Manager](/tools/ltk-manager/overview/)** - The desktop application for installing and managing mods, and the official successor to cslol-manager
- **[league-toolkit](/tools/league-toolkit/overview/)** - A Rust library for reading and writing League of Legends file formats
- **lol-meta-classes** - A toolchain for extracting and working with League's [metaclass system](/reference/metaclasses/overview/)

:::note
If you're coming from cslol-manager, it is being deprecated in favor of LTK Manager. See [Migrating from cslol-manager](/getting-started/migration/) to bring your mods over.
:::

## Who is this wiki for?

- **Mod Users** - Learn how to install and manage mods using ltk-manager
- **Mod Creators** - Discover how to create custom skins, maps, and gameplay mods
- **Tool Developers** - Find technical references for file formats and APIs
- **Contributors** - Get involved with LeagueToolkit development
