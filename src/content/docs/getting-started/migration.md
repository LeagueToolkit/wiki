---
title: Migrating from cslol-manager
description: How to migrate your mods and configuration from cslol-manager to ltk-manager.
---

If you're coming from cslol-manager, LTK Manager can import your existing mods so you don't have to reinstall them.

## Migration Wizard

On first launch (or via the migration banner in the library), LTK Manager offers to scan for cslol-manager mods.

### Steps

1. Click **Import from cslol-manager** on the migration banner
2. Browse to your cslol-manager installation directory
3. LTK Manager scans the directory and lists discovered mods
4. Select which mods to import (or select all)
5. Click **Import**

### What Happens During Import

- Each selected mod is read from the cslol-manager directory
- Mods are converted and installed in LTK Manager's mod storage
- A progress bar shows the import status
- Successfully imported mods appear in your library

### After Migration

- The original cslol-manager files are **not modified or deleted**
- Imported mods are independent copies in LTK Manager's storage
- You can dismiss the migration banner once you're done

## Dismissing the Banner

If you don't want to migrate (or have already finished), click **Dismiss** on the migration banner. This hides it permanently. The setting can be reset in the future if needed.
