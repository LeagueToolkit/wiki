A collection of helpful resources, tools, and community links for League of Legends modding.

:::tip[Looking for a tool?]
[All Tools](/tools/) documents the LeagueToolkit ecosystem with descriptions of
what each one is for. This page is the raw link list.
:::

## LeagueToolkit Projects

| Project                                                                       | Description                                                                  |
| ----------------------------------------------------------------------------- | ---------------------------------------------------------------------------- |
| [LeagueToolkit GitHub](https://github.com/LeagueToolkit)                      | The GitHub organization for all LeagueToolkit projects                       |
| [LTK Manager](https://github.com/LeagueToolkit/ltk-manager)                   | Desktop mod manager for League of Legends (Tauri + Rust + React)             |
| [LTK Manager Releases](https://github.com/LeagueToolkit/ltk-manager/releases) | Download the latest LTK Manager installer                                    |
| [league-mod](https://github.com/LeagueToolkit/league-mod)                     | CLI toolkit for creating, packing, and inspecting `.modpkg` mod projects     |
| [league-toolkit](https://github.com/LeagueToolkit/league-toolkit)             | Rust library for reading and writing League file formats                     |
| [wadtools](https://github.com/LeagueToolkit/wadtools)                         | CLI utility for extracting, listing, and comparing WAD archive files         |
| [ltk-tex-utils](https://github.com/LeagueToolkit/ltk-tex-utils)               | TEX texture CLI, plus Windows Explorer thumbnails and right-click conversion |
| [ritobin-tools](https://github.com/LeagueToolkit/ritobin-tools)               | Convert `.bin` property trees to text and back, and diff them                |
| [Mimir](https://github.com/LeagueToolkit/Mimir)                               | Compact memory-mapped hash → path tables (`.hashdb`)                         |
| [lol-meta-classes](https://github.com/LeagueToolkit/lol-meta-classes)         | Metaclass extraction toolchain for League's BIN property system              |
| [LeagueToolkit (C#)](https://github.com/LeagueToolkit/LeagueToolkit)          | The original C# library for League file format parsing                       |
