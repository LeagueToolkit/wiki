## Interactive Components

The wiki supports interactive Svelte components embedded in MDX pages. Here's an example:

<Counter client:visible label="Example Counter" />

Components use Astro's island architecture — they render statically at build time and hydrate on the client when needed. Available hydration strategies:

- `client:load` — Hydrate immediately (critical interactivity)
- `client:visible` — Hydrate when scrolled into view
- `client:idle` — Hydrate when the browser is idle
